Reviewboard Integration Release Notes
Version 1.1.0.0


v2.0.0.0
  • [Major]: Create Review now generates a single review when multiple files are selected
  • Various checks added when files are selected for review that they are in the same working copy, are source controlled etc.

v1.1.0.0
  • [Major]: Create Review option is now available when you right click .diff or .patch files
  • When a review is successfully created the Create Review tools will now automatically close
  • Jira link now uses a lowercase b so it doesn't confuse the Jira server
  • Added additional checks to the installer so we don't prompt to install software already installed
  • Reviewboard server automatically opens when the installer is complete if it's the first time it's run
  • Automatically display release notes when the installer completes if we've run the installer before

v1.0.0.0
  • [Major]: Initial release
  • [Major]: Create Review option available via right-click on SVN folder
